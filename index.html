Discord Bot: Управление Балансом и Сборами

Бот для управления балансами пользователей, сборами и штрафами в Discord.

Функционал бота

1. Управление балансом

  • Просмотр текущего баланса
  • Перевод средств между пользователями
  • История транзакций

2. Управление сборами (Parties)

  • Просмотр активных сборов
  • Присоединение и выход из сборов
  • Отправка уведомлений участникам

3. Управление штрафами

  • Наложение штрафов с автоматическим списанием средств

4. Управление посещаемостью (В разработке)

  • Статистика посещаемости участников

Команды бота

Баланс

  • !balance — показывает ваш баланс
  • !balance transfer <пользователь> <сумма> — перевод средств
  • !balance history [пользователь] — история транзакций

Примечание: команды deposit, withdraw и top доступны только для пользователей с ролью финансиста.

Сборы (Parties)

  • !party — просмотр активных сборов
  • !party join <ID сборa> — присоединение к сбору
  • !party leave <ID сборa> — выход из сборa
  • !party notify <ID сборa> <сообщение> — уведомление участников

Примечание: команды create и delete доступны только для пользователей с ролью создателя контента.

Штрафы

  • !fine <пользователь> <сумма> <причина> — наложение штрафа

Посещаемость (В разработке)

  • !attendance — список команд для работы с посещаемостью
  • !attendance top — топ участников по посещаемости
  • !attendance my — ваша посещаемость
  • !attendance member <пользователь> — посещаемость указанного пользователя

Установка и запуск

1. Клонирование репозитория

git clone https://github.com/yourusername/discord-bot.git

2. Установка зависимостей

pip install -r requirements.txt

3. Настройка конфигурации

При первом запуске автоматически создастся файл config.json со следующей базовой конфигурацией. Заполните его:

{
    "TOKEN": "YOUR_BOT_TOKEN",
    "FINE_CHANNEL_ID": 0,
    "NOTIFY_CHANNEL_ID": 0,
    "LOG_CHANNEL_ID": 0,
    "ROLE_ID": 0,
    "CONTENT_MAKER_ROLE_ID": 0,
    "FINANCIER_ROLE_ID": 0
}

4. Запуск бота

python bot.py

Часто задаваемые вопросы (FAQ)

Как узнать свой баланс?

Введите команду !balance для отображения вашего текущего баланса.

Как перевести средства другому пользователю?

Используйте команду !balance transfer <пользователь> <сумма>.

Почему у меня нет доступа к некоторым командам?

Некоторые команды доступны только для пользователей с определёнными ролями:

  • Финансисты — для управления балансом (пополнение, снятие, топ баланса).
  • Создатели контента — для управления сборами (создание, удаление сборов).

Как присоединиться к сбору?

Введите команду !party join <ID сборa> для присоединения к активному сбору.

Как работает система штрафов?

Пользователи с ролью финансиста могут использовать команду !fine для наложения штрафа, что приводит к списанию средств с баланса нарушителя и отправке уведомлений.
